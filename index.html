Cara Membeli

Ikuti langkah-langkah mudah di bawah ini untuk melakukan top up di Diamond Zone:

  1. Pilih game yang ingin Anda top up dari daftar di atas.
  2. Pilih nominal top up yang Anda inginkan.
  3. Masukkan ID Akun game Anda (jika diperlukan).
  4. Klik tombol **Pesan Sekarang** untuk diarahkan ke WhatsApp kami.
  5. Konfirmasi pesanan Anda di WhatsApp dan lakukan pembayaran.
  6. Diamond/item game akan segera masuk ke akun Anda.
